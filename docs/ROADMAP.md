# FiFi OS Roadmap (OSDev-based)

We are following the OSDev "Expanded Main Page" as the master checklist.

Sections:
1. Orientation
2. Environment and toolchain
3. Boot + hello kernel
4. CPU setup (interrupts, timers)
5. Memory management
6. Processes + scheduling + syscalls
7. Storage + filesystem
8. Userspace (init + shell)
9. GUI (framebuffer first)
10. Gaming optimization track
11. Security + privacy track
12. Custom FiFi bootloader milestone
