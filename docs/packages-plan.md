# FiFi OS Package Plan (Future Requirement)

Goal:
When FiFi OS is ready to ship as an installer image (likely a Linux-based "FiFi OS Edition" first),
it should come with default programs already installed.

Important note:
The current FiFi OS project is a custom kernel. It cannot run normal Linux packages yet.
This file is a future requirement list we will build toward.

## Categories

### 1) Security and Privacy Defaults (Base Install)
These are defensive and privacy tools/settings that reduce data collection and improve baseline security.

### 2) Gaming Defaults (Base Install)
Drivers, performance tools, compatibility layers, and QoL tooling.

### 3) Security Testing Toolkit (Optional Add-on)
If we include security testing tools, they should be an OPTIONAL install pack.
Use only on systems you own or have explicit permission to test.
